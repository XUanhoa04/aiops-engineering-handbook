# Code of Conduct

## Our pledge

We pledge to make participation in this project a harassment-free experience for everyone, regardless of age, body size, disability, ethnicity, gender identity, experience level, nationality, religion, or sexual identity.

## Our standards

**Positive behavior includes:**

- Blameless discussion of incidents and production failures
- Constructive technical critique focused on systems, not people
- Helping newcomers learn SRE / AIOps concepts
- Assuming good intent

**Unacceptable behavior includes:**

- Harassment, insults, or personal attacks
- Publishing others’ private information
- Sharing real credentials, customer data, or non-public postmortems
- Spam, off-topic promotion, or badge/star farming schemes

## Scope

This Code applies in project spaces (issues, PRs, discussions) and when representing the project publicly.

## Enforcement

Report concerns to the maintainers via GitHub issues (private contact if needed through GitHub profile). Maintainers may warn, edit, or ban contributors who violate this Code.
